Overview

User chooses from 2-3 options of study timers (Pomodoro, etc.). A description of each study method will be available.
Then, the user can start the timer and it will transition on its own (ex: 30 minute "study" timer which will end and then automatically start a 5-minute "break" timer).
Users will be able to pause or completely stop the timer and return to the main menu.

Technology Overview

Program will be hosted on my GitHub page, for anybody to use in their browser. I expect to use HTML, CSS, and JavaScript for this project.

Feature List

  • Main menu where user chooses study method or chooses "Help" where each method is explained.
  • Once user picks a method, a timer appears that can be started, paused, or stopped.
  • Option that allows user to go back to main menu.

Future Feature List

  • More aesthetic formatting.
  • Graphical clock that counts down along with textual countdown.
  • Visual or auditory cue that lets user know the timer has finished and will transition.

Basic Technical Features

  • Basic HTML.
  • CSS to enhance HTML.
  • Some JavaScript taken from Matchmaker.

Advanced Technical Features

  • Advanced JavaScript.
  • Function, if else, let, constants, switch, case, etc.
  • Advanced and more pleasing formatting with CSS.
